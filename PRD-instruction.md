# 문제 정의

기업 경영 컨설팅을 하는 모든 영업자(경영지도사, 경영 컨설턴트 등) 중 70~80%는 영업에 매진하느라 기존 고객 관리를 하지 못하고 있으며, 이러한 특성 떄문에 기존 고객의 계약이 해지되는 등의 문제를 겪는다.
특히 대표들이 원하는 정부지원사업들을 일일이 서치하지 못할 뿐더러, 특정 정부지원사업(본인이 잘 하는 것)에만 몰두하며, 다른 범위의 서비스를 제공하지 못한다.
이러한 문제점은 영업의 질을 저하시키고, 기존 고객(기업 대표들)들로부터 신뢰도가 떨어져 영업을 계속 해나가는데 지장이 생길 수 밖에 없다.
특히, 보통의 컨설턴트들은 본인의 팀을 키우는데에도 집중해야 하는데, 팀을 키우는 것보다 새로운 영업 대상을 물색하기에 급급하다.

# 타겟 이용자 및 사용 사례

## 이 제품을 사용할 핵심 사용자

기업 경영 컨설팅을 진행하는 30~60대의 컨설턴트. 영업은 잘 하지만, 기존 고객 관리가 어렵고, 다양한 분야의 정부지원사업을 알아보기에 한계가 있는 분들.

## 이 기능을 사용하는 목표

기존 고객들(기업)의 맞춤형 지원사업을 컨설턴트들에게 실시간으로 제공하여 지속적으로 새로운 정부지원사업을 제공해줄 수 있으며, 다양한 분야의 정부지원사업을 컨설턴트 본인도 배우고, 전달할 수 있도록 한다.
특히 웹사이트에 우리만의 노하우를 담은 비디오도 업로드하여 영업자들이 정부지원사업에 대한 전반적인 이해를 갖출 수 있도록 한다.
또한, 컨설턴트들은 해당 웹사이트에 본인의 고객 정보를 등록하여 통합 관리를 수월하게 할 수 있다.

# 제안 해결책

웹사이트 하나로 기존 고객(기업) 정보 등록, 실시간 맞춤 지원사업 추천, 해당 지원사업 중 기업 맞춤 지원사업 셀렉, 효율적으로 기업을 관리할 수 있도록 설계된 마이페이지에서 해당 지원사업을 기업별 목록에 넣어서 관리, 관리의 범주는 신청 진행 과정을 컨설턴트가 직접 관리할 수 있도록 하는 정도의 수준(신청 전 -> 신청 중 -> 신청 완료 -> 결과 명시(성공, 실패 등), 대시보드 페이지에서는 전반적인 정부지원사업 목록을 모두 불러와 한 번에 볼 수 있도록 한다. 또한, 다양한 연령층의 컨설턴트들이 쉽게 사용할 수 있는 직관적인 UI를 제공한다.

# 목표 및 성공 지표목표: 컨설턴트들의 기존 고객 관리의 어려움을 완화하고, 더 나은 질의 서비스를 고객에게 제공할 수 있도록 한다.

성공 지표: 론칭 후 3개월 내에 해당 웹사이트를 활용하고, 우리 회사를 알게 된 컨설턴트의 10% 가 우리 회사 이직을 희망하여, 영업 인력 확보.
